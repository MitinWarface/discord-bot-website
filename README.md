# Aurora Discord Bot

Aurora - это многофункциональный Discord бот с поддержкой музыки, системы уровней, квестов, гильдий и многого другого.

## Функциональность

### Музыкальная система
- Воспроизведение музыки из YouTube через Lavalink
- Поддержка очередей
- Команды: play, stop, skip, pause, resume, volume, loop, queue
- Управление через кнопки

### Система уровней
- Пользователи получают очки за активность
- Повышение уровня
- Таблица лидеров

### Квесты
- Система заданий с наградами
- Различные типы квестов
- Автоматическое отслеживание прогресса

### Гильдии
- Создание и управление гильдиями
- Система опыта гильдий

### Экономика
- Виртуальная валюта
- Магазин с покупками
- Система репутации
- Инвентарь

### Модерация
- Команды: ban, kick, warn
- Автомодерация
- Система предупреждений

### Тикеты
- Система поддержки через тикеты
- Управление тикетами

### Интерактивное меню
- Команда `/aurora` с интерактивными кнопками
- Простой доступ ко всем функциям бота

### События
- Система событий с уведомлениями
- Регистрация участников
- Призовые фонды

## Архитектура

### Основные компоненты

1. **index.js** - Главный файл бота, инициализация и обработка команд
2. **System/** - Подсистемы бота
   - **Audio/lavalinkSystem.js** - Музыкальная система
   - **userProfiles.js** - Система профилей пользователей
   - **guildSystem.js** - Система гильдий
   - **repSystem.js** - Система репутации
   - **eventSystem.js** - Система событий
   - **moderationSystem.js** - Система модерации
   - **notificationSystem.js** - Система уведомлений
3. **commands/** - Команды бота в формате slash-команд
4. **Config/** - Конфигурационные файлы
   - **lavalink-config.js** - Настройки Lavalink

### Музыкальная система

Музыкальная система использует Lavalink для воспроизведения аудио. Основные особенности:

- Поддержка воспроизведения из YouTube
- Управление очередью
- Поддержка перемешивания
- Контроль громкости
- Режим повтора
- Пауза и возобновление

### Команды

Все команды реализованы в формате slash-команд Discord. Основные категории:

- Музыкальные команды
- Команды модерации
- Команды экономики
- Команды для работы с профилями и квестами

## Установка и запуск

1. Установите зависимости: `npm install`
2. Настройте `.env` файл с токенами и параметрами
3. Запустите бота: `node index.js`

## Настройка Lavalink

Бот использует удаленный Lavalink сервер для воспроизведения музыки. Настройки находятся в `Config/lavalink-config.js`.
