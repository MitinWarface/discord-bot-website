# Discord Bot с Dashboard на Replit

Этот проект представляет собой Discord бота с веб-панелью управления, готовый к развертыванию на Replit.

## Особенности

- Многофункциональный Discord бот с системой уровней, квестами, магазином, гильдиями и другими возможностями
- Веб-панель управления с аутентификацией через Discord
- Возможность настройки серверов через веб-интерфейс
- Поддержка музыкальных команд (при настройке Lavalink)

## Установка на Replit

1. Создайте новый проект на Replit
2. Клонируйте этот репозиторий или скопируйте все файлы
3. В разделе "Secrets" или в файле `.env` укажите следующие переменные:

```
DISCORD_TOKEN=токен_вашего_бота
GUILD_ID=ID_основного_сервера
DISCORD_CLIENT_ID=ID_вашего_дискорд_приложения
DISCORD_CLIENT_SECRET=секрет_вашего_дискорд_приложения
BASE_URL=https://имя-вашего-проекта.repl.co
SESSION_SECRET=любой_секретный_ключ
PORT=3000
```

4. Убедитесь, что в файле `replit.nix` указана поддерживаемая версия Node.js:

```
{ pkgs }: {
  deps = [
    pkgs.nodejs-16_x
    pkgs.nodePackages.npm
    pkgs.replitPackages.jest
 ];
}
```

5. В файле `.replit` должно быть:

```
language = "javascript"

[run]
command = "npm run replit"
runInTerminal = true
```

6. Запустите проект, нажав "Run"

## Настройка Discord приложения

1. Перейдите на [Discord Developer Portal](https://discord.com/developers/applications)
2. Создайте новое приложение
3. В разделе "Bot" добавьте бота к вашему серверу
4. В разделе "OAuth2" -> "General" скопируйте Client ID и Client Secret
5. В разделе "OAuth2" -> "URL Generator" выберите:
   - Scopes: `bot`, `applications.commands`
   - Bot Permissions: `Administrator` (или необходимые права)

## Использование

После запуска бот будет работать и прослушивать команды. Веб-панель будет доступна по URL вашего Replit проекта.

## Возможные проблемы и решения

- Если бот не отвечает на команды, проверьте права на сервере и зарегистрированные slash-команды
- Если панель управления не загружается, проверьте переменные окружения
- Убедитесь, что ваш сервер Discord указан в переменной `GUILD_ID`
